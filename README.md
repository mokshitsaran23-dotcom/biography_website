# Biography Hub

Biography Hub is a static personal biography website built with HTML, CSS, and JavaScript. The project provides an engaging way to browse inspiring biographies by category, learn about famous personalities, and submit new biography entries.

## Features

- Home page with animated hero text and navigation.
- Category browsing for Scientists, Actors, Sports, Technology, Politicians, Writers, Musicians, and Artists.
- Biography submission page with form validation.
- About page describing project mission and developer information.
- Responsive navigation menu and clean layout.

## Pages

- `index.html` - Home page
- `categories.html` - Category selection for biography topics
- `submit.html` - Biography submission form with review workflow
- `about.html` - About page with mission and developer section
- `actors.html`, `artists.html`, `musicians.html`, `politicians.html`, `scientists.html`, `sports.html`, `technology.html`, `writers.html` - Individual category pages

## How to use

1. Open `index.html` in your browser.
2. Use the navigation menu to explore categories or submit a biography.
3. On the submit page, complete the form and submit the biography.

## Technologies

- HTML
- CSS
- JavaScript

## Notes

- The biography submission form validates input fields and stores submissions in memory for the current browser session.
- Image URLs are optional but should be valid if provided.
